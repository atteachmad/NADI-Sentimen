About DB-NADI

DB-NADI (Dashboard NADI) adalah sistem monitoring terpadu analitik sentimen.


Versi Sistem:
- v2.2.0 (Terupdate): Clean Mode UI Sentimen Analysis.


✖
Help Center

Cara Menggunakan DB-NADI:

  • Filter Ekosistem: Gunakan *dropdown* untuk menyaring berdasarkan kota atau kategori.
  • Geospasial Map: Klik pada *marker* di peta untuk melihat rute integrasi kompetitor dan *customer* terdekat.
